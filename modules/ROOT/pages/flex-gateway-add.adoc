To add a new gateway, follow the instructions described in the UI:

. From Anypoint Platform, select *Runtime Manager*.
. Click *Flex Gateway* in the left menu.
. Click the *Add Gateway* button.
+
image::flex-gateway-add-button.png[Add Flex Gateway button]
. Follow the instructions on the UI to add a gateway according to your OS environment.

For detailed instructions on how to add a new gateway, see xref:gateway::flex-install.adoc[Install Flex Gateway].
