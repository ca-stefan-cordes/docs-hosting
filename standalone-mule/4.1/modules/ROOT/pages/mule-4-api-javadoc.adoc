MuleSoft provides Javadocs for the Mule Kernel, the Community Edition of the Mule runtime engine.
The Javadocs contain definitions for classes, fields and their data types, and also describes methods, including their usage, parameters and return values.

Read the Mule SDK documentation if you want an easier way to extend the Mule Runtime Engine to meet your needs.
